## 数据分析报文小助手

### 简介  
数据分析报文小助手是一款基于internLM大语言模型的智能应用，旨在根据公文模板与精确数据制作可供办公人员应用的数据分析报文模板。通过先进的大模型能力，利用prompt工程抽取制作公文模板数据集，然后采用xtuner微调认知与对应能力，预期能够根据简略数据报告自动生成与人类认知相符的数据分析报文。
